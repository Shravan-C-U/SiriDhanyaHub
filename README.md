# 🌾 Siri-Dhanya Hub

A **Millet Value Chain Android App** built for Karnataka's farmers and consumers.

## 📱 About
Siri-Dhanya Hub bridges the gap between millet farmers and consumers through a single platform:
- **Farmers** get live simulated mandi prices with 7-day High/Low trends
- **Consumers** get millet recipes, health benefits, and direct FPO connect

## ✨ Features
| Feature | Description |
|---------|-------------|
| 🌾 Mandi Watch | Live simulated millet prices across Bengaluru, Davangere, Hubli |
| 🍲 Recipe Lab | Step-by-step millet recipes with search & save favourite |
| 💚 Health Benefits | Why millets are superfoods — backed by facts |
| 🛒 Direct Buy | Connect directly with Farmer Producer Organizations (FPOs) |
| 📤 Share Recipe | Share any recipe via WhatsApp or other apps |

## 🛠️ Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose + Material Design 3
- **Architecture:** MVVM
- **Local Storage:** Room Database
- **Image Loading:** Glide
- **Theme:** Earthy & Natural (Forest Green + Cream)

## 🎯 Problem Statement
Karnataka leads in millet production, but small farmers lack real-time market price visibility. Consumers are unaware of how to cook nutritious millets like Navane, Sajje, and Baragu. Siri-Dhanya Hub solves both problems in one app.

## 🌿 Impact Goals
- **Nutrition Security** — Promoting millets as superfoods
- **Climate Action** — Millets use 70% less water than paddy
- **Farmer Prosperity** — Real-time price visibility across markets
